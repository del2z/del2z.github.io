Linux常用命令

功能 命令 选项 用例
文本检索 grep -r/-R 递归检索子文件夹的文件 -n 显示行号 grep -rn 'centos' .
文本排序 sort -t 分隔符 -k 排序字段编号 -u 不打印重复记录 -n 按数值排序字段 -r 按逆序排序 sort -t$'\t' -k1n -k2nr file
字段分割 cut -d 分隔符 -f 打印字段编号 -b 打印字节编号 -c 打印字符编号 cut -d$'\t' -f2-4 infile > outfile
文件查看 cat -v 显示控制、删除及非ASCII字符 -e 行尾显示$ -t 制表符显示为^I cat -vet somefile
^ head -n K或-K 显示文件头部K行，默认为10 head -5 somefile
^ tail -n K或-K 显示文件尾部K行，默认为10 -f 跟踪显示输出 tail -5 somefile
文件编码 iconv -f 源文件编码 -t 目标文件编码 -o 输出文件 iconv -f gb18030 -t utf-8 -o tofile fromfile
远程登录 ssh -P xxx 连接端口号 ssh -P 3022 root@192.168.0.1
密钥生成 ssh-keygen ssh-keygen
信任关系建立 ssh-copy-id ssh-copy-id -i root@192.168.0.1
远程传输 scp -r 递归传输子文件夹的文件；上传：本地 -> 远程，下载：远程 -> 本地 scp -r root@192.168.0.1:/path/to/file_or_folder local_file_or_folder
文件下载 wget wget https://github.com/path/to/file
^ curl -L -O -o
文件压缩 zip -r 递归压缩子目录下的文件 zip -r archive.zip files
^ tar -c 创建压缩档 -z 用gzip压缩 -j 用bzip2压缩 -v 显示执行过程 -f 输出文件名称 tar -czvf archive.tar.gz files
文件解压缩 unzip -t 测试压缩包，不解压 unzip archive.zip
^ tar -x 解压缩 -z 用gzip压缩 -j 用bzip2压缩 -v 显示执行过程 -f 输入压缩档文件 -t 列出压缩档包含的文件 tar -xjvf archive.tar.bz2
定时任务 cron
^ crontab -u USER 设定用户的crontab服务 -e 编辑用户的crontab文件 -l 显示用户的crontab文件内容 crontab -u root cron_file
^ crontab M H d m w command [arg...]，M为分钟0~59，H为小时0~23，d为日1~31，m为月1~12，w为周0~7（0和7为星期日）
后台任务 nohup 1 标准输出 2 标准错误 /dev/null 空设备，不需要输出日志时，重定向到此处 nohup sh run.sh 1>nohup.log >&1 &
^ &
进程查看 top
^ ps -e -f aux ps -ef
进程终止 kill -6 -9
网络查看 netstat
连接测试 ping
网络设置 ifconfig

其它规则

  • 管道，将一个命令的输出传递到下一个命令的输入。
  • 重定向，将命令的标准输出/标准错误内容导入其它文件。
